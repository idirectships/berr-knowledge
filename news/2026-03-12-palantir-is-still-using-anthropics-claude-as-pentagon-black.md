# Palantir is still using Anthropic's Claude as Pentagon blacklist plays out, CEO Karp says

Palantir CEO Alex Karp confirmed the company continues using Anthropic's Claude AI despite the Pentagon labeling Anthropic a supply-chain risk. The DOD's continued reliance on Claude in the war in Iran highlights ongoing tensions between security concerns and operational needs.
