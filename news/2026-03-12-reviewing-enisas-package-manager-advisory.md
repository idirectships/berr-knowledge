# Reviewing ENISA’s Package Manager Advisory

ENISA, the EU’s cybersecurity agency, released a comprehensive technical advisory on secure use of package managers, highlighting risks and best practices for developers. The document reflects input from public consultations and underscores growing concerns around software supply chain security.
